# Тестовый проект по интеграции React приложения с Module Federation в NuxtJS v2 (на webpack4)

## Запуск приложения

- Установка зависимостей

```bash
npm run install
```

- Запуск NuxtJS и ReactJS

```bash
npm run start
```
